What to do

  1. Read the rationale.
  2. Select a phrase (drag with the mouse).
  3. The three buttons under that rationale light up. Click Quality, Context-fit, or Other (if the phrase carries a theme that is neither quality nor fit).
  4. Write a short theme label (few words) in the text field that appears, e.g., "deep interpretation" or "used patient's words".
  5. If a whole rationale fits none of the categories, click mark unclear.

Multiple highlights per rationale are fine. Same phrase can be tagged twice if it carries both judgments.
